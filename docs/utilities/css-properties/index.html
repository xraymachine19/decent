Every property a class

All CSS properties are represented by a Less variable which contains its default project setting value.

ALl CSS properties also have a class which applies that same value.

In other words, your project has:

  • a @padding-right variable available in the Less
  • a .padding-right class available in the HTML
  • a --padding-right custom property in the CSS

The point here is to reuse CSS conventions instead of adding new ones. If you know CSS, then you know most of Decent's utilities.

CSS defaults vs. project defaults

The CSS spec too has default values for elements. These are different from the project-level defaults found in properties.less. Although sometimes there can be some overlap.

The CSS default is the value displayed without any rules given to it. The project default is the intended default value for when rules are given.

The CSS default value for margin is zero. The value for @margin or .margin is probably higher. It's essentially the default for when the rule is set, vs. CSS's default for when the rule isn't.
