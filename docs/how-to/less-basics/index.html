Basics of Less

Learn how to get started with the world's best CSS preprocessor.

Decent is written in Less. Less is a preprocessor for CSS, which means it compiles and outputs to CSS.

In other words, you don't write your CSS in a CSS file. Otherwise, it's mostly the same.

Here’s an extremely quick tutorial if you don’t know it.

Variables look like this:

p {
      @variable-name: #000000;
      color: @variable-name;
  }

They’re set with the @ symbol. The above outputs:

p { color: #000000;}

Next, you can nest rules inside of each other:

p {
      @variable-name: #ffffff;
      span { 
          color: @variable-name;
      }
  }

This lets you quickly write and organize rules in a codebase. The above outputs:

p span { color: #ffffff;}

Finally, you can import other Less files:

@import "/dir/my-own-styles.less";

There you go! That covers about 90% of the Less you will ever write. It has tons and tons great features besides these, though, and great docs to go with it. But if you know CSS and the above, then you know Less. Welcome to the club!
