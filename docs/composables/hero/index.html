The hero class is usually one of the first elements on the page, outside of the normal section flow. It's full-width and heavily emphasizes all of the content within. It has a colored, patterned background.

The hero class is loud and gets attention.

Body copy should probably have a .feature class to match the size.

Normal paragraphs will be a bit bigger too, though.

As a direct descendant of the body element, the hero component takes up the full page width and commands more attention.

.hero doesn't need to be the first element, or even applied to a div. It works with other classes too.
