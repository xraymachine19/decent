composable

Examples

The .group class groups like things inside of it.

  • Apples
  • Oranges
  • Apricots
  • Watermelons
  • Kiwis
<ul class="group">
    <li>Apples</li>
    <li>Oranges</li>
    <li>Apricots</li>
    <li>Watermelons</li>
    <li>Kiwis</li>
</ul>

It's most often used for buttons and lists.

ul and ol .group composables work well with the border class:

  • Apples
  • Oranges
  • Apricots
  • Watermelons
  • Kiwis
  1. Apples
  2. Oranges
  3. Apricots

Groups work on buttons:

<ol class="group">
    <button>Mince</button>
    <button>Rinse</button>
    <button>Balance</button>
</ol>

Note: The border class works because any CSS property works as a class name, outputting the default value from properties.less.
