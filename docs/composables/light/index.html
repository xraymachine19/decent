Details

  • filename: /composables/light.less
  • settings: fill-opacity, pattern-base, output-lighter
  • defaults: none
composable · works with colors

Examples

The .light composable makes an element lighter. It works well with color classes, for easy-to-use class pairings like "light skyblue":

The only acceptable form of wrestling is gator wrestling.

<p class="skyblue padding">The only acceptable form of wrestling is gator wrestling.</p>

With the .light class, the paragraph becomes lighter.

The only acceptable form of wrestling is gator wrestling.

<p class="light skyblue padding">The only acceptable form of wrestling is gator wrestling.</p>

If the output-lighter setting is set to true, then the .lighter class will work too:

The only acceptable form of wrestling is gator wrestling.

<p class="lighter skyblue padding">The only acceptable form of wrestling is gator wrestling.</p>

This composable sets the background-image property, so it may have problems with elements that use a background-image.
