.set-contrast()

Finds a contrastful color, which is set to the variable @contrast.

Arguments

  • @background-color: A color to contrast against
  • @standard: The contrast ratio standard to meet
  • @mode: Which contrast method to use
    • tint: Add white to find contrast
    • shade: Add black to find contrast
    • auto: (default) Add whichever has more contrast potential
mixin · sets a Less variable

Examples

Basic usage

This mixin works just like .contrast(), except it sets the variable @contrast instead of outputting a property.

Since it only sets a variable, this mixin doesn't add to the CSS file:

.signup-form {
    border: 1px solid red;
    .set-contrast(red);
}

In the CSS, this doesn't output anything except the border rule that's already there:

.signup-form {
    border: 1px solid red;
}

We can overwrite the border-color with @contrast, which is available anywhere in-scope after we use .set-contrast():

.signup-form {
    border: 1px solid red;
    .set-contrast(red);
    border-color: @contrast;
}
