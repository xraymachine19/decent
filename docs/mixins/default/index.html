.default()

Takes any CSS property, and outputs a rule with the default project value for that property.

Examples

Basic usage

The .default() mixin provides a way to express a declared CSS property that uses the project default rule.

p {
    .default(border-bottom);
}

This rule would give every paragraph in the project the default border-bottom. Its output is equivalent to:

p {
    border-bottom: @border-bottom;
}

For components that need to work in multiple places, this method can help ensure the values stick to the standards.
