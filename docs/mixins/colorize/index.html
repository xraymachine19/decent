Arguments

  • @background (required): The background color
  • @colorize-contrast: Desired contrast ratio
  • @mode: Which contrast method to use
    • tint: Add white to find contrast
    • shade: Add black to find contrast
    • auto: (default) Add whichever has more contrast potential
